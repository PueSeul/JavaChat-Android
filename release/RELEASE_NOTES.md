# Java Chat v0.8.0-test

Minecraft Java 서버의 채팅과 접속자 목록을 안드로이드에서 확인하는 테스트 릴리스입니다.

## 주요 기능

- Java 1.8.9–26.2의 65개 릴리스 선택
- Microsoft 계정 및 오프라인 닉네임 접속
- 채팅, 명령어 전송과 서버 기반 자동완성
- 접속자 목록, 지연 시간과 스킨 머리 이미지
- 여러 서버 저장과 온라인 상태 확인
- 알림의 서버 나가기 버튼과 안전한 연결 종료

## 검증

- 전체 31개 자동 테스트 통과
- 공식 서버 1.8.9, 1.12.2, 1.16.5, 1.19.4, 1.21.11 통합 테스트
- Android Lint 오류 0개
- Samsung SM-S908N에서 설치 및 ViaVersion 변환기 초기화 확인
- APK Signature Scheme v2 서명 확인

테스트용 디버그 APK이므로 설치 시 출처를 알 수 없는 앱 허용이 필요할 수 있습니다.
